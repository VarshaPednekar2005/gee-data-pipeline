TO START THE SERVER WITH NEW FEATURES:

1. Open terminal
2. Run these commands:

cd /home/varshapednekar/projects/gee-data-pipeline
source venv/bin/activate
pkill -f uvicorn  # Kill old server
uvicorn clean_downloader:app --host 127.0.0.1 --port 8000 --reload

3. Open browser to: http://127.0.0.1:8000

4. Press Ctrl+Shift+R to hard refresh (clear cache)

5. Open browser console (F12) to see debug logs

6. Test with:
   - Dataset: COPERNICUS/S2_SR_HARMONIZED
   - Dates: 2024-01-01 to 2024-01-31
   - Region: Michigan
   - Click "Analyze Dataset"

7. Check console for these logs:
   - "Preview result: {..."
   - "Revisit days: 5"
   - "Creating download mode selector..."

8. Look for in the preview:
   - "Revisit Interval: 5 days"
   - "📊 Download Mode" section with radio buttons
