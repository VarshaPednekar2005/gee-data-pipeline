# GEE Data Downloader Project

## What This Project Does
Simple web interface to download satellite data from Google Earth Engine (GEE) directly to your computer.

## How It Works

### 1. GEE Connection
- Uses **Service Account Authentication** with JSON key file
- Connects to Google Earth Engine API
- No need for personal Google account authentication

### 2. Data Selection
- **Region Selection**: Choose Country, State, or Continent
- **Dataset Input**: Enter any GEE dataset ID (MODIS, Landsat, Sentinel, CHIRPS, etc.)
- **Date Range**: Set start and end dates
- **Export Format**: GeoTIFF (raster) or CSV (table)

### 3. Preview & Download
- **Preview**: Shows bands, resolution, file size, metadata
- **Visualization**: Proper color schemes for different data types
- **Direct Download**: Files download straight to your browser

## How GEE Connection Works

1. **Service Account**: Uses Google Cloud service account instead of personal login
2. **Authentication**: JSON key file provides API access
3. **Data Access**: Can access all public GEE datasets
4. **Processing**: GEE processes data in the cloud
5. **Download**: Direct download URLs generated by GEE

## Supported Datasets
- **MODIS**: Temperature, vegetation, atmosphere
- **Landsat**: Multispectral imagery (8, 9, 7, 5)
- **Sentinel**: Radar and optical data
- **CHIRPS**: Precipitation data
- **Any GEE dataset**: Just enter the dataset ID

## Export Formats
- **GeoTIFF**: Spatial raster data for GIS software
- **CSV**: Tabular data with coordinates for analysis
